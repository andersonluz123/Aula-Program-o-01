Principais termos do glossário de cloud computing

Cloud Computing
Modelo de computação que permite o acesso remoto a recursos de TI pela internet.
IAAS
Infraestrutura como Serviço, fornece recursos de computação virtualizados pela internet.
PAAS
Plataforma como Serviço, oferece um ambiente de desenvolvimento e implantação na nuvem.
SAAS
Software como Serviço, disponibiliza aplicativos pela internet, sem necessidade de instalação local.
